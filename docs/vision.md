# Meal Planner
## Purpose
The purpose of this application is to take recipes stored by individuals or households and generate a meal prep calendar. 
The intent is to simplify and automate the tedious task of having to recall the recipes the individual or households enjoy and schedule out the days where cooking is required and how many days the meal will last.
## Goals
- Make meal planning effortless and quick.
- Store favorite recipes as defined by user input and URLs.
- Allow for ultimate flexibility in planning. 
- Allow for shared use and permissions through defined roles (admin, editor, viewer).
## Non-Goals
This application will not:
- Generate recipes
- Enforce scheduling
- Assess macro or micronutrients
## Success Criteria
This project is successful if a user can:
- Have of a month of meal preparations scheduled in one click.
- Adjust the calendar as they choose
- Adjust the recipes as they choose
- Multiple people can access the same calendar
## Guiding Principles
- Offer enough funcionality without becoming overcomplicated.
- Fast interactions
- Established heirarchy of permissions
